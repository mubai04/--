# 第一章 第三扇门不能开

青岚宗停尸院在后山最阴的地方。

这里常年不见日头，石阶缝里长着青黑色的苔，风从山坳里钻进来，吹过一排排停尸木架，像有人贴着耳朵低声喘气。

陆沉舟把最后一笔写完，合上尸簿。

“周不问，药园守门人，年七十六，无修为，寿尽。”

尸簿上的字很平。

可停在木架上的人不平。

那老人瘦得只剩一层皮，双手叠在胸前，指缝里还夹着半片枯黄药叶。照规矩，这样的凡人尸身，只需停一夜，明日送去后山火窟，烧成灰，埋进杂役坟。

陆沉舟却盯着他的右手看了很久。

老人临死前，把一把钥匙塞给了他。

钥匙是黑色的，不像铁，也不像骨，摸上去温热，仿佛刚从活人心口取出来。钥匙柄上刻着三道细痕，像三扇门。

“你又在看死人发呆？”

门外传来宁小满的声音。

她端着药盘进来，袖口还沾着血，见陆沉舟没答，便走到木架边，伸手按住周不问的颈侧，又翻开眼皮看了看。

“死透了。”她说。

陆沉舟把钥匙收进袖中，问：“你确定？”

宁小满抬眼看他：“我在停尸院待了三年，见过假死、龟息、离魂、夺舍前兆。这个人没有心火，没有残魂，没有灵机回涌，连凡人最后一口阳气都散干净了。你问我确不确定？”

她顿了顿，声音低下来。

“但怪就怪在这里。”

陆沉舟看向她。

宁小满从药盘底下抽出一张旧纸，放在尸簿旁。

那也是一页停尸记录，纸角发脆，墨迹已经泛黄。

姓名一栏，写着同一个名字。

周不问。

死亡时间在十七年前。

陆沉舟的手指停在纸面上。

停尸院里很静，静得能听见木架被潮气泡胀后细微的响声。一个人死一次，是常理。死两次，就不是常理了。

宁小满看着他：“这件事不能报掌境堂。”

陆沉舟问：“为什么？”

“因为掌境堂已经在路上了。”宁小满道，“药园那边有人看见灵雾塌了。韩照骨亲自带人来。”

韩照骨。

青岚宗掌境堂副堂主，专管遗境登记、封锁和开发。宗门弟子提起他，声音都会放轻三分。因为他手里管的不是活人，是死人留下的世界。

陆沉舟低头看向木架上的老人。

周不问只是药园守门人。

至少宗门记录上是。

一个没有修为的老人，不该惊动掌境堂。

更不该把掌境堂副堂主引来。

“钥匙呢？”宁小满忽然问。

陆沉舟没有说话。

宁小满脸色变了：“他给你了？”

院外钟声响起。

一声，两声，三声。

那是掌境堂入后山的通行钟。

宁小满立刻压低声音：“交出去。陆沉舟，你只是停尸院杂役。私藏死者遗物，轻则废籍逐出宗门，重则按盗取遗境线索处置。”

陆沉舟看着周不问的尸体，忽然道：“他没有修为。”

“所以？”

“没有修为的人，不会成境。”

话音刚落，木架上的尸体消失了。

不是腐烂，不是燃烧，不是被人夺走。

那具瘦小的尸身像被无形的手从这个世界里擦掉，只剩下叠在胸前的那片枯黄药叶，轻轻落在木板上。

下一刻，整座停尸院都震了一下。

不是地震。

震动来自药园方向。

陆沉舟冲出门外，看见后山尽头的药园正在向内塌陷。不是山石坍塌，而是空间本身在收拢。药田、木屋、井台、篱笆，全都像被卷进一只看不见的壳里。

灵雾从地底翻上来，灰白如骨。

宁小满怔在原地：“遗境……”

她很快又摇头：“不对，不该这么快。稳定遗境形成前至少要三日，掌境堂要布阵、测灵、封门。现在入口在哪里？”

陆沉舟没有回答。

因为他看见了。

药园塌陷处，有一扇很窄的门。

门只有半人高，嵌在雾里，门缝中透出暗红色的光。来往杂役从旁边跑过，没有一个人停下。宁小满也看不见。

初生遗境。

尚未稳定，普通人看不见入口。

陆沉舟的袖口开始发烫。

那把黑钥匙贴着他的皮肤，一下一下跳动，像一颗不属于他的心。

掌境堂的人已经出现在山道上。

最前方那人穿着玄青法袍，腰悬白骨令，面容瘦削，眼窝很深。他只是抬头看了一眼塌陷的药园，周围弟子便全部停下。

韩照骨。

他没有看陆沉舟，却开口道：“封山。所有停尸院杂役，一个也不许离开。”

陆沉舟听见身后宁小满倒吸一口冷气。

他知道自己现在有两个选择。

交出钥匙，跪下认罪，然后等掌境堂决定他是否还有命活着离开。

或者，在入口闭合前进去。

那扇小门正在变窄。

陆沉舟握住袖中的钥匙。

就在指尖触到钥匙的一瞬间，他听见一个苍老的声音。

那声音不是从外面传来，而是从他的骨头缝里钻出来，带着濒死之人的喘息、疼痛和某种压了太久的恐惧。

“第三扇门不能开。”

陆沉舟闷哼一声，膝盖差点跪下。

剧痛从胸口炸开，沿着肋骨、脊背、左腿一路蔓延。他像在一瞬间多出了一具衰老残破的身体。那具身体曾被什么东西贯穿过，又忍着疼活了很多年。

宁小满扶住他：“你怎么了？”

陆沉舟额头全是冷汗。

他终于明白，周不问给他的不是遗物。

是门票。

山道上，韩照骨似乎察觉到什么，目光猛地转来。

陆沉舟没有再等。

他甩开宁小满的手，朝药园奔去。

“陆沉舟！”宁小满在身后喊。

掌境堂弟子也动了。

有人喝令他站住，有人拔剑，有人祭出锁灵索。陆沉舟听见风声擦过耳侧，听见锁链砸碎石阶，听见韩照骨冷冷道：“拿下。”

但他已经撞进那扇只有自己能看见的门。

天地猛地一暗。

他没有跌进药园。

脚下是黑色山门前的长阶。

阶上全是血。

陆沉舟抬起头，看见山门上挂着一块断裂的匾额。

匾额上写着三个字。

青岚宗。

可这里不是他熟悉的青岚宗。

这里的山峰被削平，剑阁坍塌，药田焦黑，外门广场上堆满尸体。那些尸体穿着青岚宗弟子的衣袍，有外门，有内门，也有掌境堂。

尸山最上方，坐着一个人。

那人还活着。

玄青法袍，白骨令，眼窝很深。

韩照骨。

陆沉舟的呼吸停了一瞬。

外面的韩照骨正在赶来。

遗境里的韩照骨却已经坐在青岚宗的尸山上，像等了他很多年。

周不问最后的声音再一次响起。

这一次，比刚才更近，近得像有个老人贴在他背后说话。

“第三扇门后，藏着青岚宗第一次灭门。”
