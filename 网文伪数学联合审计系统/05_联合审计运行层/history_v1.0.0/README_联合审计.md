# 联合审计运行说明

本目录修复了上一版“只打包、不联动”的问题。

完整输出顺序：

```text
直接测量 → 四硬闸门 → 章级六维 → 文学表现 → AI味 → 情节平滑化 → 九维风险 → AI参与伪概率 → 八维探针 → 商业六门 → 联合裁决
```

## 运行

```powershell
python 联合审计器.py self-test
python 联合审计器.py score --broad 测试样本/第一章_广域审计.json --probe 测试样本/第一章_八维探针摘要.json --output 测试报告/第一章_完整联合审计报告.json
```

同时生成同名 Markdown 报告。

## 重要边界

广域审计JSON仍需要Codex或人工按公式总表提取正文证据；本地脚本只负责校验、计算和联合裁决，不调用API。AI参与百分比是C0伪概率，不是真实作者概率。
