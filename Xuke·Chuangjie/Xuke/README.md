## 核心原则

```text
Xuke 选路径（决策）  |  Chuangjie 走路径（执行 + 模拟）
```

Xuke 不写小说、不跑模拟、不生成章节结构。  
它只回答：**在多个叙事策略中，当前应该选哪个。**

## 与 Chuangjie

Xuke 输出 `决策输出` → Chuangjie 执行与推演 → `快照更新` 回流 Xuke。
