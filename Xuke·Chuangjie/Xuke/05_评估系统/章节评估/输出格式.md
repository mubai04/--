# 章节评估输出格式

评估对象：章节序号 / 标题（由 Chuangjie 提供）

## 输出 Schema

```yaml
章节评分:
  爽感: 0-100
  节奏: 0-100
  期待: 0-100
  疲劳: 0-100      # 越高越差
  风险: 低 | 中 | 高

读者状态快照:
  期待值: 0-100
  好奇心: 0-100
  紧张感: 0-100
  爽感储备: 0-100
  疲劳值: 0-100

触发经验: []       # 本章触发的经验ID列表
告警: []           # 阈值告警信息
建议: []           # 下章策略建议
```

## 维度说明

| 维度 | 计算依据 |
|------|----------|
| 爽感 | S 变化 + 爽点类经验触发质量 |
| 节奏 | 结构阶段比例 + F 变化 |
| 期待 | Q 水平 + 期待债 |
| 疲劳 | F 绝对值 |
| 风险 | 告警等级聚合 |
