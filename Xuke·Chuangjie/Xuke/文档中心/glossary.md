# Xuke 术语表

| 术语 | 定义 |
|------|------|
| 经验原子 | 最小可复用叙事经验单元，含触发条件、结果与读者心理效果 |
| 经验链 | 多个经验原子按因果或时序组合形成的叙事规律 |
| 读者状态 | 读者在阅读过程中的心理量化模型（期待、好奇、紧张等） |
| 模式链 | 经验原子的标准组合路径，如「压制→积累→爆发→翻盘」 |
| 失败模式 | 已验证的叙事失效类型及其修复策略 |
| 拆书 | 从成品小说中逆向提取经验原子的流程 |
| 爽点 | 读者获得情绪释放与满足感的叙事节点 |
| 爽感储备 | 读者尚未释放的累积情绪势能 |
| 疲劳值 | 读者对重复套路或过长压制的耐受衰减 |
| Chuangjie | 工程执行层，负责正文、剧情、角色与世界观 |
| 决策系统 | v1.1 新增，冲突矩阵 + 权重 + 推荐引擎，输出章节策略 |
| 章节策略 | Xuke 决策输出，叙事四相计划，非正文 |
| 快照回流 | Chuangjie 完章后上报，驱动权重迭代 |
