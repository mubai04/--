## 三、三条子模式链

### 3.1 压制爆发链（爽感主线）

```
XK-YL-001 压制积累
    → XK-SJ-001 弱者成长
    → XK-GC-001 绝境爆发
    → XK-SP-001 公开翻盘
```

**适用**：打脸、逆袭、卷末高潮

### 3.2 误解崩塌链（悬念主线）

```
XK-XN-001 悬念拖延
    → XK-FZ-001 误解加深
    → XK-XN-002 信息揭晓
    → XK-SP-001 公开翻盘（可选）
```

**适用**：身份反转、阵营误会、情感线

### 3.3 弱势成长链（升级主线）

```
XK-YL-001 压制积累
    → XK-SJ-002 机缘触发
    → XK-SJ-003 挫败反思
    → XK-SJ-001 弱者成长
    → XK-GC-001 绝境爆发
```

**适用**：修炼突破、新地图开篇
