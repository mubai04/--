# 调用协议

## 系统边界

```
Chuangjie（执行层）                    Xuke（经验层）
─────────────────                    ────────────────
写正文、管剧情、管角色        ←→        决策、查询、评估、推荐
                                     不接收正文全文
                                     只接收叙事结构摘要
                                     输出章节策略（非正文）
```

## 通信格式

- 全部使用 **YAML** 块交换
- 语言：**中文**字段名
- 禁止传输：完整章节正文、角色设定全文、世界观 bible

## 允许传输

| 数据 | 示例 |
|------|------|
| 叙事结构摘要 | 「本章压制为主，末段小释放」 |
| 触发经验ID | `["XK-YL-001"]` |
| 读者状态 | 五维数值 |
| 章节元数据 | 序号、字数、类型 |

## 调用顺序（v1.1 建议）

```
1. 规划阶段 → 决策接口（完整策略输出）
2. 写作阶段 → Chuangjie 内部执行
3. 完章阶段 → 评估接口 + 快照更新
4. 下章规划 → 决策接口（含历史快照回流）
5. 轻量查询 → 查询接口 / 推荐接口
```

## 错误处理

| 错误 | 处理 |
|------|------|
| 经验ID 不存在 | 返回空，记录日志 |
| 读者状态缺字段 | 拒绝请求，列出缺项 |
| 触发经验与状态矛盾 | 返回告警，不阻断 |
