# 查询接口

## 用途

Chuangjie 按条件检索经验原子与模式链。

## 请求

```yaml
操作: 查询
类型: 经验原子 | 模式链 | 失败模式
条件:
  类别: []           # 可选
  标签: []           # 可选
  经验ID: []         # 可选，精确
  状态: 已发布       # 默认只返回已发布
```

## 响应

```yaml
结果数: 0
条目:
  - 经验ID: ""
    名称: ""
    路径: "02_经验资产库/..."
```

## 实现说明（v1）

v1 为文件系统实现：Agent 或脚本按路径 glob + YAML 解析即可，无需 API 服务。
