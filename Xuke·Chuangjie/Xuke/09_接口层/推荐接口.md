# 推荐接口

## 用途

根据当前读者状态，推荐应触发的经验或模式链。  
**v1.1 起**：完整决策请用 `决策接口.md`；本接口保留轻量查询。

## 请求

```yaml
操作: 推荐
协议版本: "1.1"

读者状态:
  期待值: 0-100
  好奇心: 0-100
  紧张感: 0-100
  爽感储备: 0-100
  疲劳值: 0-100

上下文:
  已用经验: []        # 最近5章经验ID
  章节序号: 0
  章节阶段: 起势 | 压制 | 转折 | 高潮 | 收束
```

## 响应

```yaml
推荐:
  - 类型: 经验原子 | 模式链
    ID: ""
    名称: ""
    理由: ""
    最终得分: 0
    优先级: 1-10

告警: []
降级策略: {}
```

## 推荐规则（v1.1 · 权重引擎摘要）

| 条件 | 推荐 | 禁止 |
|------|------|------|
| S ≥ 70 | XK-GC-001, XK-SP-001 | XK-YL-001 |
| F ≥ 70 | XK-SP-001, XK-SJ-003 | XK-YL-001 |
| H ≤ 25 | XK-XN-001, XK-FZ-001 | — |
| T ≥ 80 且 S ≥ 50 | XK-GC-001 | — |
| 最近2章已用 YL-001 | — | XK-YL-001（降权） |
| 同卷第2次 GC-001 | XK-SP-001 | XK-GC-001 |

完整公式见 `10_决策系统/经验权重函数.md`。  
冲突校验见 `10_决策系统/经验冲突矩阵.yaml`。
