# 经验ID 映射表

Chuangjie 调用时可使用 **XK 标准ID** 或 **别名（EXP_*）**，二者等价。

| XK 标准ID | 别名 | 名称 |
|-----------|------|------|
| XK-YL-001 | EXP_PRESSURE_ACCUMULATION | 压制积累 |
| XK-SP-001 | EXP_FACE_SLAP_PUBLIC | 权威面前公开翻盘 |
| XK-XN-001 | EXP_SUSPENSE_DELAY | 悬念拖延 |
| XK-SJ-001 | EXP_WEAK_TO_STRONG | 弱者成长 |
| XK-QX-001 | EXP_EMOTION_OUTBURST | 情绪爆发 |
| XK-FZ-001 | EXP_MISUNDERSTANDING | 误解加深 |
| XK-SJ-002 | EXP_OPPORTUNITY_GAIN | 机缘触发 |
| XK-XN-002 | EXP_INFO_REVEAL | 信息揭晓 |
| XK-SJ-003 | EXP_FAILURE_REFLECTION | 挫败反思 |
| XK-GC-001 | EXP_CRITICAL_BREAKTHROUGH | 绝境爆发 |

## 命名规则

- **XK-{类别缩写}-{序号}**：系统内唯一主键，入库必选
- **别名**：外部系统兼容标识，单条经验最多一个

类别缩写：YL=压力 SP=爽点 XN=悬念 SJ=升级 QX=情绪 FZ=反转 GC=高潮
