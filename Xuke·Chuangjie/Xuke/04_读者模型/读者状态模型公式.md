# 读者状态模型公式

> 本质：不是写故事，是操控阅读体验曲线。
> 所有数值范围：**0–100**（变化量见经验原子中的 ± 字段）。

---

## 一、状态变量定义

| 变量 | 符号 | 含义 |
|------|------|------|
| 期待值 | Q | 读者对后续发展的预期强度 |
| 好奇心 | H | 对未知信息的探索欲 |
| 紧张感 | T | 危机感与不确定性压力 |
| 爽感储备 | S | 尚未释放的累积情绪势能 |
| 疲劳值 | F | 对重复与冗长的耐受衰减 |

---

## 二、初始状态（开书 / 新卷）

```
Q₀ = 50
H₀ = 40
T₀ = 20
S₀ = 0
F₀ = 0
```

可根据品类微调 ±10，但单变量不得超过 [20, 80] 区间。

---

## 三、章节更新公式

每读完一个叙事单元（章），按经验原子叠加变化量，再应用衰减与边界：

### 3.1 叠加

```
Q' = Q + Σ(经验.读者心理效果.期待值)
H' = H + Σ(经验.读者心理效果.好奇心)
T' = T + Σ(经验.读者心理效果.紧张感)
S' = S + Σ(经验.读者心理效果.爽感储备)
F' = F + Σ(经验.读者心理效果.疲劳值)
```

### 3.2 自然衰减（每章末尾）

```
H  = H' × 0.92        # 好奇心随时间自然回落，需新钩子补充
T  = T' × 0.95        # 紧张感略降，除非章末钩子抬高
F  = min(100, F' × 1.02 + 章节数×0.5)   # 疲劳缓慢累积
```

### 3.3 边界钳制

```
Q, H, T, S, F = clamp(值, 0, 100)
```

### 3.4 爽感释放联动

当本章触发 **爽点类经验**（如 XK-SP-001）且 S > 50：

```
实际爽感 = min(S, 基础释放量) × (1 - F/200)
S = S - 实际爽感
F = max(0, F - 实际爽感×0.3)    # 有效释放可部分缓解疲劳
```

## 四、阈值与告警

| 条件 | 等级 | 建议动作 |
|------|------|----------|
| F > 70 | 危险 | 换手法、插支线、提前释放（XK-YL-001） |
| S > 80 且 F > 50 | 警告 | 必须安排释放，否则爆发被稀释 |
| H < 20 连续 2 章 | 警告 | 投放新悬念或钩子（XK-XN-002） |
| Q > 85 且未兑现 3 章 | 危险 | 期待债，需实质性推进 |
| T > 85 连续 2 章 | 警告 | 需缓冲章或部分释放 |

## 五、综合阅读体验指数（章节级）

用于 `05_评估系统` 输出参考：

```
阅读体验 = 0.25×Q + 0.20×H + 0.15×T + 0.25×S_eff + 0.15×(100-F)

其中 S_eff = S × (1 - F/150)   # 疲劳稀释下的有效爽感储备
```

输出范围 0–100，建议：

- **≥ 75**：健康
- **55–74**：需关注
- **< 55**：介入调整
