# 沉浸中断因子

识别导致读者「出戏」的叙事行为，每章累计 `中断分`。

## 中断事件（示例）

| 事件 | 中断分 |
|------|--------|
| 逻辑硬伤被读者识别 | +25 |
| 角色 OOC 无解释 | +20 |
| 节奏突然注水 | +15 |
| 信息重复解释 | +10 |
| 视角混乱 | +15 |

## 对状态的影响

```
若中断分 > 30：
  F += 中断分 × 0.5
  Q -= 中断分 × 0.3
  沉浸度 = 100 - 中断分（供评估系统引用）
```

沉浸度 < 60 时，建议 Chuangjie 下章优先修复逻辑或节奏，再推剧情。
