# 系统边界规范

> 工业级原则：**决策与执行不可耦合**（Decision/Execution Decoupling）

---

```text
Xuke 负责「选路径」
Chuangjie 负责「走路径」
```

## Xuke 禁止事项

- 运行故事 / 跑剧情因果链
- 生成章节结构草案
- 执行章节推演模拟
- 输出正文

## Chuangjie 禁止事项

- 自动选经验原子
- 自动选模式链
- 权重计算 / 冲突决策
- 叙事策略推荐
- 修改经验资产库

**若在 Chuangjie 内实现上述能力 → 形成「双大脑」，系统无法收敛。**

## 数据流向

```
Xuke.决策输出  ──只读──→  Chuangjie
Chuangjie.快照更新  ──只写──→  Xuke.Snapshot快照层
```

Chuangjie 不得反向写入 Xuke 经验库；仅通过 `学习信号` 建议迭代。
