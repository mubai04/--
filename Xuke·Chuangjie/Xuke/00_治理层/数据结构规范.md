# 数据结构规范

## 通用约定

- 所有结构化数据使用 **YAML**
- 编码：**UTF-8**
- 缩进：**2 空格**
- 必填字段不可省略；可选字段可省略或置空
- 列表项使用 `-` 开头

## Schema 与实例分离

```
01_知识模型层/经验原子.schema.yaml   → 定义字段与类型
02_经验资产库/爽点/XK-SP-001.yaml    → 具体经验实例
```

## 字段类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 字符串 | 单行文本 | `"压制过久必须释放"` |
| 多行文本 | 描述性内容 | 前置条件说明 |
| 整数 | 0–100 评分类 | `82` |
| 枚举 | 固定选项 | `低` / `中` / `高` |
| 列表 | 字符串数组 | 标签、原因 |
| 嵌套对象 | 子结构 | 读者状态 |

## 引用规范

- 经验链引用经验原子：使用 `经验ID` 列表
- 模式链引用：使用相对路径或 ID，禁止硬编码剧情

## 校验

新增或修改实例前，须符合 `01_知识模型层` 对应 Schema。
