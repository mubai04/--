# 命名规范

## 目录命名

- 使用中文 + 数字前缀排序：`00_治理层`、`02_经验资产库`
- 子目录用中文名词：`爽点/`、`情绪模型/`

## 文件命名

| 类型 | 格式 | 示例 |
|------|------|------|
| 规范文档 | `{主题}.md` | `架构规范.md` |
| Schema | `{实体名}.schema.yaml` | `经验原子.schema.yaml` |
| 经验原子 | `{经验ID}.yaml` | `XK-SP-001.yaml` |
| 模式链 | `{模式名}.yaml` | `压制爆发链.yaml` |
| 接口文档 | `{接口名}.md` | `查询接口.md` |

## 经验ID 编码

```
XK-{类别缩写}-{三位序号}

类别缩写：
  SP = 爽点    XN = 悬念    YL = 压力
  FZ = 反转    SJ = 升级    QX = 情绪
  GC = 高潮    JJ = 结局    TY = 通用
```

## 版本号

- 经验原子：`主.次`（如 `1.0`、`1.1`）
- 系统整体：见 `版本管理规则.md`
