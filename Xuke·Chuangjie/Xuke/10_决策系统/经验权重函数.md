# 经验权重函数

> 决定「用不用、用哪个、用哪条链」  
> 版本：**1.1**

---

## 一、总公式

对每条候选经验原子 `a`：

```
最终得分 = 基础匹配分 × 读者适配系数 × 阶段系数 × 历史惩罚系数 − 冲突惩罚分
```

得分范围钳制：`clamp(最终得分, 0, 100)`

---

## 二、基础匹配分（0–100）

检查经验原子的 `前置条件` 与 `触发条件`：

```
基础匹配分 = 50
  + (满足前置条件数 / 前置条件总数) × 25
  + (满足触发条件数 / 触发条件总数) × 25
```

若任一**硬性触发条件**不满足（如「S > 60」而当前 S=40）→ 基础匹配分 = 0，不进入候选。

---

## 三、读者适配系数（0.0–1.5）

根据当前读者状态 `(Q, H, T, S, F)` 与经验 `读者心理效果` 方向是否一致：

```
适配 = 1.0
  + 0.01 × (经验.期待值 × sign(Q缺口))
  + 0.01 × (经验.好奇心 × sign(H缺口))
  + ...
```

**缺口 sign 规则**（系统需补什么）：

| 状态 | 缺口方向 | 优先经验类别 |
|------|----------|--------------|
| S > 70 | 需释放 | 爽点、高潮 |
| F > 60 | 需缓解 | 爽点、升级（反思缓冲） |
| H < 30 | 需悬念 | 悬念、反转 |
| T > 75 且 S > 50 | 需爆发 | 高潮、爽点 |
| T < 30 且 Q < 50 | 需加压 | 压力、反转 |
| Q > 80 连续未兑现 | 需推进 | 悬念揭晓、翻盘 |

**快速查表（v1.1 硬编码）**：

| 条件 | 加权经验ID | 系数 |
|------|------------|------|
| S ≥ 70 | XK-GC-001, XK-SP-001 | ×1.4 |
| F ≥ 70 | XK-SP-001, XK-SJ-003 | ×1.3 |
| H ≤ 25 | XK-XN-001, XK-FZ-001 | ×1.3 |
| T ≥ 80 且 S ≥ 50 | XK-GC-001 | ×1.5 |
| S ≤ 30 且 章节阶段=对峙 | XK-YL-001 | ×1.2 |
| 最近2章已用 XK-YL-001 | XK-YL-001 | ×0.5 |
| 最近1章已用 XK-GC-001 | XK-GC-001 | ×0.2 |

读者适配系数 = 查表系数之积，上限 1.5，下限 0.2。

---

## 四、阶段系数（0.5–1.3）

| 章节阶段 | 加权类别 | 系数 |
|----------|----------|------|
| 起势 | 压力、悬念 | 1.2 |
| 压制 | 压力、升级 | 1.3 |
| 转折 | 反转、机缘、悬念 | 1.2 |
| 高潮 | 高潮、爽点、情绪 | 1.3 |
| 收束 | 悬念揭晓、升级反思 | 1.1 |

阶段与经验类别映射：

```
起势   → YL, XN
压制   → YL, SJ
转折   → FZ, SJ-002, XN
高潮   → GC, SP, QX
收束   → XN-002, SJ-003
```

不匹配当前阶段 → 阶段系数 = 0.6

---

## 五、历史惩罚系数（0.3–1.0）

```
历史惩罚 = 1.0
  − 0.3 × (该原子在最近5章出现次数 − 1)    # 重复惩罚
  − 0.2 × (该原子在 failed_patterns 中出现)  # 失败记录惩罚
```

下限 0.3。

---

## 六、冲突惩罚分（0–100）

来自 `经验冲突矩阵.yaml`：

```
冲突惩罚 = max(原子对冲突.严重度) + avg(读者状态冲突.严重度) × 0.5
```

若 `同章允许: false` 且被同章组合 → 冲突惩罚 += 50

---

## 七、模式链权重

模式链得分 = 链内各步**下一待执行步**的原子最终得分之加权平均：

```
模式链得分 = Σ(步得分 × 步权重) / Σ步权重
步权重: 当前应执行步 = 1.0，后续步 = 0.3
```

---

## 八、Top3 选取规则

1. 按最终得分降序排列所有候选原子
2. 剔除冲突惩罚 ≥ 70 的组合
3. 取 Top3 原子，**两两之间不得同章禁止**
4. 模式链取 Top1，且与 Top3 原子不重复占满同章槽位

---

## 九、熔断 override（优先级最高）

熔断规则覆盖权重，强制推荐：

| 熔断条件 | 强制推荐 | 禁止 |
|----------|----------|------|
| F > 70 | XK-SP-001 或 XK-SJ-003 | XK-YL-001 |
| S > 80 | XK-GC-001 或 XK-SP-001 | XK-YL-001, XK-XN-001 |
| H < 20 连续2章 | XK-XN-001 或 XK-FZ-001 | XK-SJ-003 |
| 同卷 GC-001 第2次 | XK-SP-001 | XK-GC-001 |

详见 `06_失败系统/` 与 `10_决策系统/经验冲突矩阵.yaml`。
