# Xuke v2 · 决策系统协议（Spec）

> 版本：**2.0**（v1.1 已合并入 v2，字段兼容）  
> 状态：**可运行协议层**  
> 定位：叙事经验操作系统 · **自动决策引擎**（仅 Xuke，不属于 Chuangjie）

**完整 v2 规范见：`v2决策协议.md`**
---

## 一、系统边界（已冻结）

### Xuke = 经验层 + 决策层

```
Xuke
├─ 02_经验资产库      经验原子（已入库 10 条）
├─ 03_模式系统        经验链 / 模式链
├─ 04_读者模型        S / T / F / Q / H 公式
├─ 06_失败系统        失败模式 + 熔断
├─ 09_接口层          对外调用协议
└─ 10_决策系统        ← 本层：冲突 / 权重 / 推荐 / 快照
```

### Chuangjie = 执行层（未动）

```
Chuangjie
├─ 章节生成
├─ 角色行为执行
├─ 世界状态更新
├─ Plot 调度
└─ Snapshot 快照
```

**Xuke 输出章节策略（结构），不输出正文。**

## 三、决策流水线（六层）

| 序号 | 层名 | 输入 | 输出 | 规范文件 |
|------|------|------|------|----------|
| 1 | 输入层 | 章节上下文 + 读者状态 + 历史 | 标准化请求 | `模板/01_输入层.schema.yaml` |
| 2 | 候选层 | 输入层 | 候选原子 + 模式链 | `模板/02_候选层.schema.yaml` |
| 3 | 冲突层 | 候选层 | 冲突得分 + 读者风险 | `经验冲突矩阵.yaml` |
| 4 | 权重层 | 冲突层 + 读者状态 | 最终优先级 | `经验权重函数.md` |
| 5 | 决策层 | 权重层 | Top3 策略 + 叙事计划 | `模板/05_决策输出层.schema.yaml` |
| 6 | 快照层 | 执行结果 | 学习信号 | `Snapshot快照层/snapshot.schema.yaml` |

## 四、符号约定

| 符号 | 字段名 | 范围 |
|------|--------|------|
| Q | 期待值 | 0–100 |
| H | 好奇心 | 0–100 |
| T | 紧张感 | 0–100 |
| S | 爽感储备 | 0–100 |
| F | 疲劳值 | 0–100 |

变化量使用 ±整数，与经验原子 `读者心理效果` 一致。
