## 三、核心公式（v2 定版）

### 3.1 经验原子最终得分

```
Score(a) = clamp(
    Match(a) × ReaderFit(a) × Stage(a) × History(a) − Conflict(a),
    0, 100
)
```

| 因子 | 范围 | 规范 |
|------|------|------|
| Match | 0–100 | 前置 + 触发条件满足率 |
| ReaderFit | 0.2–1.5 | 读者状态缺口查表 |
| Stage | 0.6–1.3 | 章节阶段 × 经验类别 |
| History | 0.3–1.0 | 近期重复使用惩罚 |
| Conflict | 0–100 | 冲突矩阵 |

### 3.2 模式链得分

```
Score(chain) = Σ Score(下一步) × W_step / Σ W_step
W_step: 当前步=1.0, 后续步=0.3
```

### 3.3 读者曲线预测

```
ΔQ = Σ 经验.期待值 + 衰减(Q)
ΔH = Σ 经验.好奇心 × 0.92
ΔT = Σ 经验.紧张感 × 0.95
ΔS = Σ 经验.爽感储备（含释放联动）
ΔF = Σ 经验.疲劳值 × 1.02
```

完整衰减与释放：`04_读者模型/读者状态模型公式.md`

### 3.4 冲突聚合

```
Conflict = max(原子对严重度) + avg(读者状态冲突) × 0.5
若同章禁止组合: Conflict += 50
决策阈值: <40 通过 | 40–69 警告 | ≥70 拒绝
```

规范：`经验冲突矩阵.yaml`

## 四、推荐排序算法

完整逐步算法：`推荐引擎/推荐排序算法.md`

摘要：

```
1. 熔断检查（override，最高优先级）
2. 候选筛选（Match > 0）
3. 冲突评估
4. 权重计算 → 降序排列
5. Top3 贪心选取（两两非同章禁止）
6. 模式链 Top1
7. 组装叙事四相 + 读者曲线预测 + 风险警告
```

## 五、熔断规则（override）

| 条件 | 强制推荐 | 禁止 |
|------|----------|------|
| F > 70 | XK-SP-001, XK-SJ-003 | XK-YL-001 |
| S > 80 | XK-GC-001, XK-SP-001 | XK-YL-001, XK-XN-001 |
| H < 20 连续2章 | XK-XN-001, XK-FZ-001 | — |
| 同卷 GC-001 第2次 | XK-SP-001 | XK-GC-001 |

熔断优先于一切权重计算。
