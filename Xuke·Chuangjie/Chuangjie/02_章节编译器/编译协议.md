# 章节编译协议（Narrative Compiler）

> 02_章节编译器 · 把策略翻译成剧情结构

---

## 输入

`01_策略接收层` 输出的 `章节策略`

---

## 编译规则

将叙事四相映射为六段剧情结构：

```text
压制段 → 冲突触发段 → 危机扩大段 → 转折窗口 → 爆发段 → 收束段
```

| 四相（Xuke） | 编译段 |
|--------------|--------|
| 压制相 | 压制段 |
| 触发相 | 冲突触发段 + 危机扩大段 |
| 爆发相 | 转折窗口 + 爆发段 |
| 收束相 | 收束段 |

---

## 输出

```yaml
章节结构:
  章节ID: ""
  章节阶段: ""
  编译段:
    压制段: []
    冲突触发段: []
    危机扩大段: []
    转折窗口: []
    爆发段: []
    收束段: []
  触发经验: []
  主模式链: ""
  核心冲突: []
  角色节点: []
```

Schema 继承：`01_知识模型层/章节结构.schema.yaml`（兼容叙事四相字段）

---

## 约束

- 每段 1–5 条结构节点，每条 ≤ 50 字
- 禁止正文段落
- 每段须映射 ≥1 经验ID
