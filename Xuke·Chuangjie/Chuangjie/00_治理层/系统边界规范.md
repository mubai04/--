# 系统边界规范

## 禁止事项（Decision/Execution 耦合防护）

Chuangjie **不得**实现以下能力（否则形成双大脑）：

- 自动选经验原子 / 模式链
- 权重计算 / 冲突决策
- 叙事策略推荐 / 排序
- 复制 `Xuke/10_决策系统/` 六层模板

Chuangjie **只能**只读消费 `Xuke.决策输出`。

## 禁止跨界

| 禁止行为 | 归属 |
|----------|------|
| 在 Xuke 内跑章节推演 | Chuangjie `06_读者模拟器` + `05_叙事物理引擎` |
| 在 Xuke 内生成章节结构 | Chuangjie `02_章节编译器` |
| 在 Chuangjie 内修改经验原子 | Xuke `02_经验资产库` |
| 在 Chuangjie 内做策略决策 | Xuke `10_决策系统` |

## 数据流向（单向原则）

```
Xuke.决策输出  →  Chuangjie（只读消费）
Chuangjie.快照更新  →  Xuke（只写快照层）
```

Chuangjie **不得**反向修改 Xuke 经验库；仅通过快照 `学习信号` 建议迭代。

## 读者模型公式

- **定义权**：Xuke `04_读者模型/`
- **仿真应用权**：Chuangjie `06_读者模拟器/` 引用并执行公式
- **物理计算权**：Chuangjie `05_叙事物理引擎/` 维护世界运行时六维
- **禁止**：Chuangjie 自建另一套读者心理字段名

## 语言规范

与 Xuke 一致：目录、字段、文档 **全中文**。
