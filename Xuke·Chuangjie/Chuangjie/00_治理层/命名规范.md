# 命名规范

## 目录命名

与 Xuke 对齐：`00_治理层`、`06_叙事模拟器`

## 文件命名

| 类型 | 格式 | 示例 |
|------|------|------|
| Schema | `{实体名}.schema.yaml` | `章节结构.schema.yaml` |
| 模板 | `{层名}.template.yaml` / `.json` | `模拟输入.template.yaml` |
| 快照记录 | `{章节ID}.yaml` | `CH-018.yaml` |

## ID 编码

| 类型 | 格式 | 示例 |
|------|------|------|
| 章节 | `CH-{序号}` | `CH-018` |
| 角色 | `JS-{序号}` | `JS-001` |
| 剧情线 | `PL-{序号}` | `PL-001` |
| 仿真运行 | `SIM-{章节ID}-{轮次}` | `SIM-CH-018-01` |

## 与 Xuke ID 引用

- 经验ID：直接使用 Xuke 的 `XK-*`
- 模式链ID：直接使用 Xuke 的 `XL-*`
- 禁止在 Chuangjie 内重新定义经验别名为主键
