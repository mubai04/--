# 策略接收协议

> 01_策略接收层 · 消费 Xuke v2 决策输出

---

## 输入（来自 Xuke）

- 主模式链（XL-*）
- 支撑经验（XK-* Top3）
- 叙事四相计划
- 读者曲线预测
- 风险警告 + 降级策略

规范：`../11_接口层/接收Xuke决策.md`

---

## 输出：章节策略对象

```yaml
章节策略:
  协议版本: "2.0"
  章节ID: ""
  主模式链: ""
  支撑经验: []
  目标读者曲线:
    期待值: 0      # 变化量预测
    好奇心: 0
    紧张感: 0
    爽感储备: 0
    疲劳值: 0
  叙事四相:
    压制相: []
    触发相: []
    爆发相: []
    收束相: []
  风险警告:
    等级: 低 | 中 | 高
    说明: []
```

---

## 校验

- [ ] 经验ID 存在于 Xuke 资产库
- [ ] 模式链ID 有效
- [ ] 协议版本 ≥ 2.0
- [ ] **不在本层做决策或权重计算**
