# Chuangjie v2.0 — 叙事运行时引擎（NRT Engine）

**Chuangjie = 把叙事策略运行成剧情世界的小说物理引擎**

```text
Xuke 选路径  |  Chuangjie 走路径 + 可计算物理仿真
```

## 十层架构

```
01 策略接收 → 02 章节编译 → 03 世界运行 → 04 角色行为
→ 05 叙事物理 → 06 读者模拟 → 07 冲突推进 → 08 章节输出
→ 09 快照 → 10 反馈回流 Xuke
```
