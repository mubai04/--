# 叙事物理公式

> Chuangjie v2.0 · 可计算叙事引擎核心  
> 归属：**05_叙事物理引擎**（执行层，非 Xuke 决策层）

---

## 零、公式化本质

```text
叙事 = 状态变化 + 冲突传播 + 节奏调节 + 情绪传导 + 读者反馈
```

本引擎维护 **世界运行时六维**；读者五维由 `06_读者模拟器` 引用 Xuke 公式计算。

---

## 一、指标定义

| 符号 | 字段名 | 范围 | 定义 |
|------|--------|------|------|
| P | 压制值 | 0–100 | 角色/局势承受的压力累积 |
| SD | 爽点强度 | 0–100 | 本章可释放的满足峰值（非 Xuke 爽感储备 S） |
| X | 悬念值 | 0–100 | 未解信息与读者期待的势能 |
| C | 冲突强度 | 0–100 | 内外冲突总量，驱动事件推进 |
| E | 情绪波动 | −50–+50 | 负=消沉，正=激动 |
| R | 章节风险 | 0–100 | 节奏崩坏 / 读者掉线风险 |

**读者五维**（与 Xuke 对齐）：期待值 Q、好奇心 H、紧张感 T、爽感储备 S、疲劳值 F  
—— 由 §3 从世界六维 + 经验原子映射得出。

## 二、核心传播公式

### 2.1 压制 → 爆发

```
爆发概率 BP = P / (100 − C + 1)

爆发强度 BI = clamp(SD + P × α1 − max(0, −E) × α2, 0, 100)
```

- `α1` = 经验原子权重（来自 Xuke 决策 `支撑经验`，默认 0.6）
- `α2` = 情绪阻尼（默认 0.4）
- **规则**：压制越久 → 爆发潜力越大；情绪消沉 → 爆发减弱

触发条件：`BP > 0.65` 且 `P > 50` → 标记爆发段

---

### 2.2 悬念递增

```
X_next = clamp(X_current + Σ(ΔX_事件) − β × F_reader, 0, 100)
```

- `ΔX_事件`：每个悬念事件 +5～+15（由事件类型查表）
- `β` = 悬念衰减系数（默认 0.3）
- `F_reader` = 读者疲劳值（来自仿真前状态）
- **规则**：信息隐藏越久 → X 越高；疲劳高 → 悬念失效加速

---

### 2.3 冲突推进

```
C_next = clamp(C_current + 基础冲突 + (P/100)×α_c + 矛盾密度×β_c + 目标冲突 − δ×缓冲段长度, 0, 100)
```

- `C_事件` / 基础冲突：+3～+20
- `α_c=0.4`, `β_c=0.3`（默认）
- `δ` = 缓冲消减系数（默认 2.5 / 段）
- **规则**：高冲突 → 高张力；须周期性缓冲防 R 爆表

### 2.3b 张力/节奏（Sigmoid）

```
T_pace = σ(k × (C/100 − 0.5)) × 100 − 节奏瓶颈 + (X/100)×10

σ(x) = 1 / (1 + e^(−x))，k 默认 6
```

连续高压无缓冲 → `节奏瓶颈 = 15`

---

### 2.4 世界不稳定度（03 世界运行引擎）

```
不稳定度 U = w_p × (P/100) + w_c × (C/100) + w_u × 外部事件系数

默认: w_p=0.35, w_c=0.45, w_u=0.20
```

`U > 0.7` → 触发世界观级连锁事件候选

---

### 2.5 情绪传播（04 角色行为系统）

```
E_next = clamp(E_current + Σ(ΔE_行为) + κ × (C/100 − P/100), −50, 50)
```

- `κ` = 压力-冲突耦合（默认 15）
- 角色压迫越强 → E 向负或正极端偏移 → 行为越极端

---

### 2.6 爽点强度累积

```
SD_next = clamp(SD_current + (P × η1) − (F_reader × η2), 0, 100)
```

- `η1=0.4`, `η2=0.25`
- 高压制 + 低疲劳 → SD 累积；为爆发段储备

## 三、读者五维映射（06 读者模拟器）

叙事物理态 → 读者状态变化（章级）：

```
ΔQ = γ_q × (X/100 × 20) + γ_q2 × (BI/100 × 15)
ΔH = γ_h × (X/100 × 25) − γ_h2 × (F/100 × 10)
ΔT = γ_t × (C/100 × 30) + γ_t2 × (P/100 × 10)
ΔS = γ_s × (SD/100 × 40) − γ_s2 × (BI/100 × 35)   # 爆发时 S 释放
ΔF = γ_f × (P/100 × 15) + γ_f2 × 重复套路计数 × 5
```

**默认系数**（可由 Xuke 经验原子 `读者心理效果` 覆盖）：

| 系数 | 默认 |
|------|------|
| γ_q | 1.0 |
| γ_h | 1.0 |
| γ_t | 1.0 |
| γ_s | 1.0 |
| γ_f | 1.0 |

叠加后应用 Xuke `04_读者模型/读者状态模型公式.md` 的衰减规则。

## 四、冲突推进器（07）调节公式

```
if T < 30:  C_adj = C + 15          # 张力过低，加冲突
if F > 70:  P_adj = P − 20          # 疲劳过高，降压制
if H < 25:  X_adj = X + 20          # 好奇不足，加悬念
if S > 80:  触发爆发段标记           # 须释放
```

调节仅修改 **结构稿建议**，不回写 Xuke 决策；若调节幅度大 → `10_反馈回流层` 标记「建议重决策」。

## 五、章节风险熔断

```
R = w1×(P/100) + w2×(C/100) + w3×(|E|/50) + w4×(F/100)

默认: w1=0.25, w2=0.30, w3=0.20, w4=0.25
```

| R 区间 | 等级 | 动作 |
|--------|------|------|
| < 40 | 低 | 正常输出 |
| 40–69 | 中 | 07 冲突推进器介入 |
| ≥ 70 | 高 | 熔断：请求 Xuke 重新决策 |
