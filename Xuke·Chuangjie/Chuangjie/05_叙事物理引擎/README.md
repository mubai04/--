# 叙事物理引擎（Narrative Physics）

> 05_叙事物理引擎 · Chuangjie 可计算核心

---

## 职责

控制四类传播：

| 传播类型 | 规则摘要 |
|----------|----------|
| 冲突传播 | C 随事件累加，缓冲段消减 |
| 信息传播 | X 随隐藏/揭露变化 |
| 节奏传播 | P ↔ SD ↔ 爆发段 |
| 情绪传播 | E 随角色行为偏移 |

---

## 核心规则（自然语言）

- 压制越久 → 爆发越强
- 信息隐藏越久 → 悬念越高（疲劳加速衰减）
- 角色压迫越强 → 行为越极端
- 冲突过高且无缓冲 → R 熔断
