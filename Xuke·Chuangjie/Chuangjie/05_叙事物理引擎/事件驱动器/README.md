# 事件驱动器（Event Driver）

> 对应双系统架构图 · Chuangjie 执行引擎 ⑥  
> 归属：`05_叙事物理引擎` 子模块

---

## 职责

在叙事物理计算完成后，**触发具体剧情事件**，推动 Plot 向前：

- 根据因果链与冲突强度，激活编译段中的事件节点
- 更新世界状态 delta
- 为读者模拟器提供「已发生事件」序列

---

## 输入 / 输出

```
输入: 因果链 + 冲突强度 C + 编译段事件图
输出: 已触发事件[] + 世界状态变化 + 触发经验ID确认
```

---

## 与相邻模块

| 模块 | 关系 |
|------|------|
| 因果引擎 | 上游：提供触发链 |
| 07_冲突推进器 | 下游：根据模拟结果调节下一批事件密度 |
| 06_读者模拟器 | 消费：已触发事件 → 读者五维变化 |

---

## 边界

事件驱动 **不选择策略**（不选经验原子）；只执行已编译结构中的事件触发逻辑。
