# 反馈回流层（Feedback Loop）

> 10_反馈回流层 · 闭环核心

---

## 职责

```text
章节结果 → 组装快照更新 → 推送 Xuke
```

更新 Xuke 侧：

- 经验原子权重（历史惩罚系数）
- 模式链成功率
- 失败模式候选
- 读者模型参数建议（α, β, γ, w）

---

## 输出格式

对齐 Xuke `10_决策系统/Snapshot快照层/snapshot.schema.yaml`：

```yaml
快照更新:
  章节ID: ""
  执行策略: { 模式链, 使用经验 }
  结果: { 读者状态变化, 成功得分 }
  学习信号: { 加权经验, 降权经验, 新模式候选 }
```

规范：`../11_接口层/回流Xuke快照.md`

---

## 重决策触发

当 `07_冲突推进器.是否建议重决策 == true` 或 `章节风险 R ≥ 70`：

```yaml
回流扩展:
  重决策请求: true
  原因: ""
  建议调整: []
```

---

## 边界

- **只写** Xuke 快照层，**不改**经验库
- 学习信号为建议，由 Xuke `08_训练进化系统` 审核入库
