# 快照系统（Snapshot System）

> 09_快照系统

---

## 记录内容

- 世界状态 + 世界运行时六维
- 角色状态
- 冲突状态
- 读者五维
- 章节结构稿
- 仿真ID / 偏差分

---

## 目录

```
09_快照系统/
├─ 记录/           # {章节ID}.yaml
├─ 索引.yaml       # 最近 N 章
└─ schema.yaml
```

---

## Schema

```yaml
快照:
  章节ID: ""
  仿真ID: ""
  世界运行时: { 压制值, 爽点强度, 悬念值, 冲突强度, 情绪波动, 章节风险 }
  读者状态: { 期待值, 好奇心, 紧张感, 爽感储备, 疲劳值 }
  章节结构: { ... }
  成功得分: 0
  时间戳: ""
```
