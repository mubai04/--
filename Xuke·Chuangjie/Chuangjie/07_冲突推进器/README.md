# 冲突推进器（Conflict Driver）

> 07_冲突推进器 · 自动调节节奏，防止剧情「死掉」

---

## 逻辑

| 条件 | 调节动作 |
|------|----------|
| 紧张感 T < 30 | 冲突强度 C +15 |
| 疲劳值 F > 70 | 压制值 P −20 |
| 好奇心 H < 25 | 悬念值 X +20 |
| 爽感储备 S > 80 | 标记须爆发段 |

公式：`05_叙事物理引擎/叙事物理公式.md` §4

---

## 输出

```yaml
节奏调节:
  调节项: []
  结构稿修正建议: []
  是否建议重决策: false    # R ≥ 70 时为 true
```

---

## 边界

调节 **结构稿**，不修改 Xuke 决策。  
`是否建议重决策: true` → 触发 `10_反馈回流层` 向 Xuke 发重决策请求。
