# 章节推演模拟接口

> Chuangjie 核心接口 — 不属于 Xuke

---

## 用途

对已生成的**章节结构草案**运行叙事物理引擎 + 读者反应仿真。

## 请求

```yaml
操作: 章节推演
协议版本: "1.0"

模拟输入:
  仿真ID: "SIM-{章节ID}-{轮次}"
  运行模式: 预演 | 复盘 | 联调
  来源策略: { ... }      # 来自 Xuke 决策输出摘要
  章节结构: { ... }      # 来自 02_章节执行层
  仿真前读者状态: { ... }
  世界状态: { ... }      # 可选
```

完整 Schema：`06_叙事模拟器/模板/01_模拟输入.schema.yaml`

## 响应

```yaml
协议版本: "1.0"

模拟输出:
  仿真后读者状态: { ... }
  读者状态变化: { ... }
  偏差分析:
    偏差分: 0-100
    结论: 准确 | 中等偏差 | 策略失效
  仿真结论:
    是否建议调整策略: true | false
    是否可进入正文: true | false

快照更新: { ... }        # 自动填充，可直推 Xuke
```

完整 Schema：`06_叙事模拟器/模板/03_模拟输出.schema.yaml`

## 执行步骤

1. 叙事物理引擎 → `仿真运行态`
2. 读者反应仿真（引用 Xuke 读者模型公式）
3. 偏差分析（对比 `来源策略.读者曲线预测`）
4. 组装 `快照更新`

示例：`06_叙事模拟器/模板/示例/章节推演示例.yaml`

---

## 边界提醒

本接口 **只在 Chuangjie 内调用**。  
Xuke 不得实现本接口。
