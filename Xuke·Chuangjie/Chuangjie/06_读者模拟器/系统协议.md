# 读者模拟器 · 系统协议

> 06_读者模拟器 · 执行层模拟（非 Xuke 决策）

---

## 定位

⚠️ **执行层模拟**：对已生成章节结构跑读者反应，不是决策。

回答：**如果这样写，读者 Q/H/T/S/F 会怎样变化？**

---

## 输入

- 章节结构（编译段）
- 叙事物理引擎输出（P, SD, X, C, E, R）
- 仿真前读者状态
- Xuke 经验原子 `读者心理效果`

---

## 输出

```yaml
读者仿真:
  期待值变化: 0
  好奇心变化: 0
  紧张感变化: 0
  爽感储备变化: 0
  疲劳值变化: 0
  掉线风险: 低 | 中 | 高
  偏差分析:
    偏差分: 0
    结论: 准确 | 中等偏差 | 策略失效
```

---

## 计算公式

1. 叙事物理 → 读者五维映射：`05_叙事物理引擎/叙事物理公式.md` §3
2. 叠加 Xuke 经验原子效果
3. 应用 Xuke 衰减规则：`../../Xuke/04_读者模型/读者状态模型公式.md`
4. 对比 Xuke `目标读者曲线` → 偏差分
