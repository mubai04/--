# 世界运行引擎（World Runtime）

> 03_世界运行引擎 · 维护世界状态

---

## 职责

- 势力/阵营平衡
- 资源变化
- 地点状态
- 时间推进
- 事件触发器

---

## 输出

```yaml
世界状态:
  阵营平衡: ""
  资源: []
  地点状态: []
  时间标记: ""
  不稳定度: 0.0    # 见 05_叙事物理公式 §2.4
  章节delta: {}
```

---

## 与叙事物理的关系

每章开始时加载快照中的 `世界运行时`；  
章节仿真结束后写回 delta。

公式：`05_叙事物理引擎/叙事物理公式.md` §2.4
