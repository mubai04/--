# 章节执行层 · 执行协议

> 将 Xuke `决策输出` 转化为 `章节结构草案`

---

## 输入

Xuke `09_接口层/决策接口.md` 响应中的 `决策输出` 块。

## 输出

符合 `01_知识模型层/章节结构.schema.yaml` 的结构草案。

## 映射规则

| Xuke 决策字段 | Chuangjie 章节结构字段 |
|---------------|------------------------|
| `选定策略.主模式链` | `主模式链` |
| `选定策略.支撑经验` | `触发经验` |
| `叙事计划.压制相` | `叙事四相.压制相` |
| `叙事计划.触发相` | `叙事四相.触发相` |
| `叙事计划.爆发相` | `叙事四相.爆发相` |
| `叙事计划.收束相` | `叙事四相.收束相` |

## 执行约束

1. 结构节点每条 ≤ 50 字，禁止写正文段落
2. 每条结构节点须可映射到至少一个经验ID
3. 同章 `触发经验` 须通过 Xuke 冲突矩阵校验

## 下一步

结构草案 → `06_叙事模拟器` 预演 → 通过后可选进入正文生成
