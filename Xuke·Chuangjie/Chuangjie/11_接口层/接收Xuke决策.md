# 接收 Xuke 决策接口

> Chuangjie 消费 Xuke 策略的入口

---

## 调用方式

Chuangjie 在章节规划阶段向 Xuke 发起：

```yaml
操作: 决策
协议版本: "1.1"
输入:
  章节上下文: { ... }
  读者状态: { ... }
  历史快照: { ... }
```

## 响应消费

收到 `决策输出` 后：

1. `02_章节执行层` 按 `执行协议.md` 映射为 `章节结构草案`
2. `06_读者模拟器` 预演
3. 通过后可选进入正文生成

## 规范引用

- Xuke 决策接口：`../Xuke/09_接口层/决策接口.md`
- Xuke 决策示例：`../Xuke/10_决策系统/模板/示例/决策运行示例.yaml`

## 重新决策触发

当模拟输出 `偏差分析.结论 == 策略失效` 或 `偏差分 > 30`：

```yaml
操作: 决策
输入:
  章节上下文:
    剧情意图: "仿真偏差过大，请求调整"
  历史快照:
    失败模式: ["SIM-CH-018-01"]
```
