# 正文目录规则

## 唯一真源

唯一允许人工编辑的正文真源：

```text
正文/作品/<WORK_ID>/当前版本/章节/
```

任务副本、合并文本、报告引用、缓存和 Codex 临时修改稿均不是正文真源。

## 章节命名

```text
0001_章节标题.md
0002_章节标题.md
0010_章节标题.md
```

要求：

- 四位数字前缀；
- `.md` 或 `.txt`；
- 按数字前缀排序；
- 同一作品不得出现重复章节号；
- 所有文本统一 UTF-8；
- 正文目录不得放报告、设定表或提示词。

## 版本

- `当前版本/章节/`：唯一当前正文；
- `历史版本/<VERSION>/章节/`：只读快照；
- 版本比较必须保留两版正文 SHA-256；
- 不得以“最终版2”“最新版”等模糊名称代替版本号。

## 整书分析

整书任务禁止静默截断。超出当前处理上限时必须：

1. 拒绝整书结论；或
2. 明确改为连续章节/片段任务；
3. 在报告中记录实际范围。
