# 反刷分协议 v0.2

每次提取必须完成六项检查：

- `AG_SENSORY_STUFFING`：机械堆感官；
- `AG_RANDOM_REVERSAL`：无因果反转；
- `AG_MEANINGLESS_CHOICE`：无后果选择；
- `AG_SYMBOL_SPAM`：符号重复刷记忆点；
- `AG_RANDOM_RHYTHM`：随机断句刷节律；
- `AG_HARMFUL_OMISSION`：以留白名义隐藏必要信息。

状态：`PASS / FAIL / UNKNOWN`。

任何 FAIL 都禁止 `ROBUST_PASS`，并至少进入 `TARGETED_REPAIR`。UNKNOWN 必须在报告中保留，不得自动视为通过。

自动测试必须覆盖：

1. 单指标极端输入；
2. 同一证据重复支撑；
3. 非法数值；
4. 短文本虚假 PASS；
5. 多指标冲突；
6. 权重扰动 Stability。
