# 项目范围

## 项目目标

将网文文本中的可核验事实、描述比例、代理风险和人工判断转换为可审计的分析报告，并把不确定判断路由为确定修复动作。

## 包含

- 正文目录治理；
- 单章、连续章节和整书范围任务；
- F0 事实、F1 比率、F2 九维风险；
- 六维章节能力；
- 三态硬闸门；
- 证据位置与文件校验；
- Stability；
- 版本比较；
- 批处理；
- 反刷分检查。

## 不包含

- 自动生成小说正文；
- 真实作者身份鉴定；
- 平台留存、推荐或商业成功预测；
- 服务器、数据库、多人权限和 API 调用；
- C0 权重的外部统计结论。

## 正文唯一真源

`正文/作品/<WORK_ID>/当前版本/章节/`

分析任务、报告、缓存和引用均不得成为第二正文真源。
